HTML : Hypertext Markup Language

  • Hypertext refers to the way webpages are linked together. Thus, the links available in webpages are called Hypertext.
  • As this language simply "marks-up" a text document with tag that tells a web browser how to structure it to display, it is called a Markup Language.
  • It is the the backbone of a webpage and widely used throughout the world to create web pages.
  • It is a not a case sensative language.


HTML Elements

A HTML element consist of a content enclosed in a start tag and an end tag.
SYNTAX : <tag>content</tag>


HTML Attribute

A HTML attribute is a key-value pair that is present in start tag of an element and provides additional information about the element.
SYNTAX : name = "value"


Headings

  • They are the titles or subtitles that are dispalyed in the page.
  • They are defined from <h1> to <h6> tags.
  • Search engines use headings to index the structure and content of a webpage.

Paragraphs

  • It always starts on a new line and usually is a block of text.
  • The browser automatically discards any extra spaces.

Division

  • The <div> tag defines a container for an element in a HTML document which is styled or manipulated with javascript.
  • It is styled using a class or id attribute.
  • By default, a browser leaves a line break before and after <div&> tag.


Comments

  • These are the texts used for documentation purpose only and not displayed in the browser.
  • SYNTAX : <!-- This is a comment -->
