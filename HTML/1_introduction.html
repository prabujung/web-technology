HTML : Hypertext Markup Language

  • Hypertext refers to the way webpages are linked together. Thus, the links available in webpages are called Hypertext.
  • As this language simply "marks-up" a text document with tag that tells a web browser how to structure it to display, it is called a Markup Language.
  • It is the the backbone of a webpage and widely used throughout the world to create web pages.
  • It is a not a case sensative language.


HTML Elements

A HTML element consist of a content enclosed in a start tag and an end tag.
SYNTAX : <tag>content</tag>


HTML Attribute

A HTML attribute is a key-value pair that is present in start tag of an element and provides additional information about the element.
SYNTAX : name = "value"
