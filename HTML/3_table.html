Table

  • Allows developers to arrange data in row and column.
  • <tr> - defines table row.
  • <td> - defines table data.
  • <th> - defines table head.
  • <caption> - defines table caption.
  • <thead> - defines table head.
  • <tbody> - defines table body.
  • <tfoot> - defines table footer.
