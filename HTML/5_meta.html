Meta

The <meta> tag typically used to specify the character set, page description, keywords, author of the document, and viewport settings.
The metadata will not be displayed on the page, but is used by browsers (how to display content or reload page), by search engines (keywords), and other web services.
SYNTAX : <meta name="..." content="...">



Audio

This tag is used to play an audio file in web browser.
SYNTAX : 

                
                    <audio>
                        <source src="path" type="audio/mpeg">
                    </audio>
The different attributes used in audio tag are listed below.
  • autoplay : automatically plays the audio when the page loads
  • preload : specifies how the audio should be loaded (auto/metadata/none) when the page loads
  • loop : plays the audio file over and over again



Video

This tag is used to show video on webpage.
