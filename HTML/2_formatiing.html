Text Formatting

  • <b> : This text is bold.
  • <strong> : This text is important.
  • <i> : This text is itallic.
  • <em> : This text is emphasised.
  • <small> : This text is smaller.
  • <sup> : a2+b2
  • <sub> : H2O
  • <mark> : This text is highlighted.
  • <del> : This text is striked through.
  • <ins> : This text is inserted and underlined.

Spacing

  • <br> : It denotes a line break in a document.
  • <pre> : It is used to display preformatted text. For instance: 
                        I'm sittin on a toilet
                    I had to take a dodee
                    When the dodee hit the water
                    the water hit my b**ty
  • &nbsp; : It adds  a space in the document.
  • &ensp; : It adds  two spaces in the document.
  • &emsp; : It adds  four spaces in the document.

Text phrase formatting

  • <span> : It is an inline container used to mark-up a part of a document.
  • <tt> : This tag was used in HTML4 to define teletype text and is not supported in HTML5.

Image element

  • <img> is used to embed elements in a HTML document.
  • Images are not technically inserted but linked to a document.
  • <img> tag created a holding space for the referenced image.
  • SYNTAX : <img src="hello.jpg" alt="picture of a hand">

Anchor

  • The anchor tag <a> is used to create a hyperlink in our document.
  • href attribute is used to define the address of another document.
  • target attribute defines where to open the document.

List

  • List is used to group set of related items in a list.
  • There are three types : ordered <ol>, unordered <ul> and description <dl>.
  • <li> tag is used to add a list item in both ordered and unordered list whereas <dt> and <dd> is used in description list.
  • <dt> defines term whereas <dd> describes the term.
